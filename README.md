## рџљЂ Setup Guide

### 1. Prerequisites
```bash
# Install Flutter (3.16+)
flutter --version

# Install Firebase CLI
npm install -g firebase-tools

# Install FlutterFire CLI
dart pub global activate flutterfire_cli
```

### 2. Create Firebase Project
1. Go to [Firebase Console](https://console.firebase.google.com)
2. Create a new project named `campusbondhu`
3. Enable the following services:
   - **Authentication** в†’ Email/Password provider
   - **Cloud Firestore** в†’ Start in production mode
   - **Storage** в†’ Default bucket

### 3. Configure Flutter with Firebase
```bash
# Login to Firebase
firebase login

# Link your Flutter app
flutterfire configure --project=YOUR_FIREBASE_PROJECT_ID
```
This generates `lib/firebase_options.dart` automatically.

### 4. Deploy Security Rules
```bash
# Deploy Firestore rules
firebase deploy --only firestore:rules

# Deploy Storage rules
firebase deploy --only storage
```

### 5. Install Dependencies & Run
```bash
flutter pub get
flutter run
```
