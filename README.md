# 🎓 CampusBondhu  
### Student Interest & Collaboration Platform

CampusBondhu is a full-stack mobile platform designed to connect university students across Bangladesh based on **interests, skills, academic goals, and project ideas**.

Unlike traditional campus apps, CampusBondhu focuses on **interest-based networking, collaboration, and peer-driven learning**, enabling students to build meaningful academic and social connections beyond departmental boundaries.

---

## 🚀 Vision

To create a **collaborative student ecosystem** where learners can:
- Find study partners
- Build projects together
- Share knowledge and resources
- Discover opportunities across campuses

---

## 🧩 Core Features (Planned System)

### 👤 1. User Profile (Passion Map)
Each user creates a profile containing:
- Name, University, Department, Year
- Interests (e.g., Programming, Photography)
- Skills (e.g., Python, UI Design)
- Bio / Goals

📌 Purpose:
> Enables intelligent matching between students

---

### 📚 2. Study Buddy Matching
- Find students studying the same subject
- Send/accept buddy requests
- Create study groups

📌 Future Enhancement:
- AI-based matching using similarity scores

---

### 🎨 3. Hobby Match
- Connect students based on shared interests
- Discover hobby communities (music, sports, tech)

---

### 🚀 4. Project Launchpad
- Post project ideas
- Join existing projects
- Form teams with complementary skills

📌 Example:
> “Looking for Flutter + ML teammate for health app”

---

### 📰 5. Campus Community Feed (HallWire)
- Share posts, ideas, and discussions
- Engage with comments and reactions
- Cross-campus communication

---

### 🛒 6. Campus Marketplace
- Buy/sell academic items:
  - Books
  - Notes
  - Devices
- Location-based listings

---

### ❓ 7. Freshman Guide (Q&A System)
- Juniors ask questions
- Seniors provide guidance
- Knowledge sharing ecosystem

---

### 📅 8. Event Discovery
- Browse campus events
- Post new events
- Get notifications

---
